Statistical analysis using SPSS software

SPSS is a widely used software for statistical analysis and data management. It can be used for descriptive statistics, inferential statistics, data visualization, and more.

Structural Equation Modeling using AMOS

EM is a statistical technique used to test complex relationships among multiple variables, with AMOS is a powerful software that allows the researcher to test hypotheses about latent variables and their relationships with observed variables.

Econometric analysis using STATA

STATA is a software package that provides a wide range of econometric and statistical tools for data analysis. It can be used for panel data analysis, time-series analysis, and more.

Data visualization using R

R is a programming language and software environment for statistical computing and graphics. It can be used to create interactive visualizations of data, such as line plots, bar charts, scatter plots, and more.

Qualitative data analysis using NVivo

NVivo is a qualitative data analysis software that can be used to organize, analyze, and make sense of unstructured data, such as interview transcripts, survey responses, and social media data.
